How to Use

1. Place Aspects: Drag aspects from the library onto empty hexes to set endpoints. Mobile: tap an aspect, then tap a hex.

2. Add/Remove Gaps: Right-click an empty hex to toggle it as a void. Mobile: tap an empty hex with nothing selected.

3. Use More: Check "Use More" under an aspect to make the algorithm prefer using it.

4. Enable/Disable: Uncheck an aspect to exclude it from the solve.

5. Solve: Click "Research!" to automatically find the shortest parent-child path connecting your placed aspects. Don't like the layout? Click it again for a different valid pattern.

6. Edit Formulas: Click the ⚙️ icon next to a compound aspect to edit its parents.

7. Reset Hexes: Clears all placed aspects and gaps from the grid.

8. Reset Changes: Clears custom aspects, enabled/disabled aspects, and grid size back to defaults.

Note: With many endpoints, the solver can sometimes route through the same aspect via two separate branches instead of merging them into one shared path -- a known quirk that doesn't affect correctness. This project is open source -- check it out on GitHub.
