How to Use

1. Place Aspects: Drag aspects from the left library and drop them onto the empty hexes to set your starting endpoints. On mobile: tap an aspect to select it, then tap a hex to place it.

2. Add/Remove Gaps: Right-click an empty active hex to turn it into an inactive void (gap). Right-click again to restore it. On mobile: press and hold a hex instead.

3. Use More: Check "Use More" under an aspect to make the algorithm prefer using it.

4. Solve: Click "Research!" to automatically find the shortest parent-child path connecting your placed aspects.

5. Edit Formulas: Click the ⚙️ icon next to a compound aspect to edit its parents.
